A Wordpress content slider with delayed animations and background parallax effect.

== Description ==

Vina Parallax Content Slider Widget for Wordpress is a simple parallax content slider with different animations for each slider element and a background parallax effect. Using CSS animations, we'll control the animation of each single element in the slider and create a parallax effect by animating the background of the slider itself. It saves you space and makes for a better user experience.

Features:

- Support multi Wordpress Version.
- Get title, description from post item.
- Load image from Feature Images.
- Load post items from a specific category.
- Allows to adjust the number, ordering of post item displayed. 
- Allows to show/hide title, image, description and readmore button.
- Allows resize image before display.
- Languages included: English
- Allow loading wordpress widget inside post, page.
- To manipulate almost everything !
- Totally customizable and skinnable using HTML and CSS
- Lightweight, modern and fast-loading design.
- W3C XHTML 1.0 Transitional. W3C CSS Valid.
- Tableless design and 100% CSS-based
- Fully compatible IE8+, Firefox 10+, Flock 0.7+, Netscape, Safari, Opera 9.5
- Easy installation

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire 'vina-parallax-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'Vina Parallax Content Slider Widget' to your sidebar
